Register: 建立帳戶，其中Type中必須輸入Seller或Buyer。登入完後會先回首頁，並為登出狀態。2. Login: 登入創建的帳號3. 登入後可以看到上方橫幅內有username，點入可以查看帳戶資訊及修改帳戶資訊。4. 買家帳號只能進行下單，而賣家帳號能新增商品，但不能下單5. AddCart是新增到購物清單，進入MyCart並Order才是下訂單。訂單資訊會加入MyOrder
